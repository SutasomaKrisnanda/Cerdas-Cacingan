# **App Name**: Cerdas Cacingan

## Core Features:

- Title Scene: Welcome screen with title, illustration, and 'Start' button.
- Cacingan Explanation Scene: Explains what Cacingan is with interactive pop-up buttons.
- Transmission Scene: Interactive scene where users click on images to learn how worms enter the body.
- Worm Types Scene: Scene introducing different types of worms with clickable info cards.
- Symptoms Scene: Describes the symptoms of Cacingan, with clickable icons.
- CERDAS Prevention Scene: Interactive scene where each letter of 'CERDAS' reveals a prevention tip.
- Closing Scene: Concluding screen with encouraging message and 'Repeat' button.

## Style Guidelines:

- Primary color: Bright pink (#F186AB) for buttons and titles to evoke fun and energy.
- Background color: Light blue (#9ED8F3), a desaturated, brighter hue creating a cheerful backdrop.
- Accent color: Light green (#C4E53B), providing a contrasting color for information and secondary actions.
- Font: 'Inter', a sans-serif font, to give the application a rounded and bold style which is perfect for children.
- Single-page application layout with full-screen scenes and 'next' and 'back' buttons.
- Interactive elements: pop-up modals and clickable cards with rounded corners.
- Use illustrations and emojis suitable for children for enhanced comprehension.
